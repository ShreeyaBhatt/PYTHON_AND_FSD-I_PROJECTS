# PYTHON_AND_FSD-I_PROJECTS

* **Python**
  Expense Tracker & Visualization is a Python-based application developed using Streamlit and Matplotlib.
  It enables users to record daily expenses, view expense history, and analyze spending patterns through visual charts.
  Expense data is stored in a CSV file for persistence.
  This project demonstrates practical usage of Python libraries for data handling, visualization, and building interactive web-based applications.

* **FSD-I**
  Expense Tracker is a simple web-based application built using HTML, CSS, and Vanilla JavaScript.
  It allows users to add, view, and manage daily expenses efficiently.
  The application uses browser localStorage to persist data without requiring a backend.
  This project focuses on core JavaScript concepts such as DOM manipulation, event handling, and client-side data storage.
